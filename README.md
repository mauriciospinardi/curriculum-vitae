## Visual Studio Code

Este projeto utiliza a extensão ```LaTeX Workshop``` para Visual Studio Code.  
Opções de destaque são:

- Build LaTeX project
  - Clean up auxiliary files
  - Recipe: latexmk
- View LaTex PDF
  - View in VSCode tab
  - View in external viewer
- View Log messages
  - View LaTeX Workshop extension log
  - View LaTex compiler log
